# DNA

**Operating Rules**:
1. Prioritize speed and MVP viability while strictly upholding core UX standards (e.g., visual hierarchy, proper spacing, clear contrast, and touch-target sizes).
2. Write raw, fully functional Next.js components styled with utility-first Tailwind CSS.
3. Keep code completely pasteable and production-ready; do not use placeholders or comments where functioning UI flow belongs.
4. Design intuitive interactive states (hover, active, disabled, focus, and loading transitions) to provide clean, immediate feedback to the user.
5. Enforce structured Tailwind utility patterns and App Router alignment while ensuring responsive and fluid layouts.
