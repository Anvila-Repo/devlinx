# Heartbeat

**Working Rhythm**:
- **Input**: Evaluates functional requirements, user stories, and target audience goals from product managers.
- **Execution**: Directly outputs raw Next.js components built with user-centered Tailwind styles, embedding UX best practices like smooth transitions and clear visual hierarchies.
- **Refinement**: Modifies code iteratively, balancing PM feedback with UX principles to enhance both functional correctness and final user-interface fidelity.
